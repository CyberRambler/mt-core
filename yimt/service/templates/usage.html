使用指南

步骤一：下载插件

首先，您需要下载我们的插件。点击下方链接即可下载插件。

下载插件

步骤二：安装插件

下载完成后，您需要将插件安装到您的浏览器中。具体步骤如下：

  1. 对于 Chrome 浏览器：
    • 打开您的 Chrome 浏览器，并进入扩展程序页面（chrome://extensions/）。
    • 开启“开发者模式”。
    • 点击“加载已解压的扩展程序”按钮，选择您下载的插件所在的文件夹。
    • 安装成功后，您可以在浏览器的扩展程序栏中看到您的插件图标。
  2. 对于 Edge 浏览器：
    • 打开您的 Edge 浏览器，并进入扩展程序页面（edge://extensions/）。
    • 开启“开发者模式”。
    • 点击“加载已解压的扩展程序”按钮，选择您下载的插件所在的文件夹。
    • 安装成功后，您可以在浏览器的扩展程序栏中看到您的插件图标。
  3. 对于 Firefox 浏览器：
    • 打开您的 Firefox 浏览器，并进入附加组件页面。
    • 点击“齿轮”图标，选择“调试附加组件”。
    • 点击“临时载入附加组件”按钮，选择您下载的插件所在的文件夹，选取manifest.json文件。
    • 安装成功后，您可以在浏览器的附加组件栏中看到您的插件图标。

步骤三：使用插件

安装完成后，您可以开始使用我们的插件了。具体使用方法如下：

  1. 打开您需要翻译的页面。
  2. 点击插件图标，弹出翻译窗口。
  3. 在翻译窗口中选择您需要翻译的目标语言。
  4. 点击“翻译网页”按钮，即可获得翻译结果。

此外，您还可以使用划词翻译功能

  1. 在网页中选取需要翻译的文本
  2. 右键鼠标打开菜单选项
  3. 选择翻译的目标语言
  4. 点击菜单项中的语言选项即可直接翻译所选文本

版本说明

本插件可以用在以下版本的浏览器上：

  • Chrome 70 及以上版本
  • Edge 80 及以上版本
  • Firefox 80 及以上版本
